Read All the words(Ascending order):

./ssort input.txt 100 ALL ASC

Read All the words(Descending order):

./ssort input.txt 100 ALL DESC

Read ALPHA the words(Ascending order):

./ssort input.txt 100 ALPHA ASC

Read ALPHANUM the words(Ascending order):

./ssort input.txt 100 ALPHANUM ASC

Read ALPHA the words(Ascending order) AND skip the word 'firas':

./ssort input.txt 100 ALPHA ASC firas
